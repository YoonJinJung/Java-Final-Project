# 주차장 관리 프로그램

## 소개
이 프로젝트는 2학년 2학기 자바 프로그래밍 실습의 기말 프로젝트로 만들어진 주차장 관리 프로그램입니다. 100% JAVA 언어와 Swing 라이브러리를 사용하여 개발되었습니다.

## 주요 기능
1. 주차 공간 할당 및 반환
2. 현재 주차 상태 확인
3. 주차 요금 계산
